## Installing

```
$ git clone git@github.com:a4t/misudo.git

or

$ wget -P /tmp https://github.com/a4t/misudo/releases/download/v0.0.1/misudo-v0.0.1-linux-amd64.zip
$ unzip misudo-v0.0.1-linux-amd64.zip
```

## Build

```
$ make deps
$ make

or

$ make cross-build
```
